## How To Start

Clone this repo
``` shell
git clone https://github.com/BrunoEmaSN/calendar-app.git
```

Get into the directory. Edit  environment variables with you credentials

## Running the app

```shell
# Install dependencies
npm install

# Run the application
npm start

# Run the tests
npm run test

# Build
npm run build
```
